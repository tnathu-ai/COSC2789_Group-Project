how to run the dash board
- To run the dash board, you will go to the cmd of the directory ./COSC2789/Dash
- After that you will write the command line: python app.py
- copy the address and paste it into the browswer
- incase you want to have new dataset, you need to input it into the assignment3.ipynb and run all the cells
- after running the notebook, there will be an update csv file call 'cleaned_data.csv' in Dash folder
- You can repeat step 1
